Linux әлеміне қош келдіңіз

Бұл сайтта сіз төрт әйгілі Linux дистрибутивімен танысасыз. Әрқайсысы өзіндік философиясы мен артықшылықтарына ие — таңдауыңызға еркіндік!

Бастау

Ubuntu

Ubuntu — қарапайым әрі тұрақты жүйе, жаңа қолданушылар үшін ең танымал Linux нұсқасы. Оның басты ұраны — «Humanity to others». Орнату оңай, интерфейсі таза және жаңартулары тұрақты түрде шығып отырады.

Келесі

Kali Linux

Kali Linux — этикалық хакерлер мен қауіпсіздік мамандарына арналған дистрибутив. Ішінде Nmap, Metasploit, Wireshark және басқа жүздеген құрал бар. Қауіпсіздік зерттеулері мен пентест үшін мінсіз.

“The quieter you become, the more you are able to hear.”
Келесі

Fedora

Fedora — жаңа технологияларды тез енгізетін, инновациялық Linux платформасы. Ол ашық кодтың алдыңғы шебінде тұрады және тұрақтылық пен қауіпсіздікті біріктіреді.

Келесі

Arch Linux

Arch Linux — еркіндік пен бақылау философиясын ұстанады. Әр қолданушы өз жүйесін нөлден құрастырады. Rolling release моделі арқылы әрқашан жаңарып отырады.
